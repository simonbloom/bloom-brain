# Setup Agent - Environment Configuration

Comprehensive environment setup and configuration agent providing interactive setup guidance and automation for: **$ARGUMENTS**

## Agent Purpose
This agent provides comprehensive environment setup guidance, including MCP server configuration, environment variables, tool configuration, and automated setup validation with step-by-step instructions for optimal development environment preparation.

## Analysis Scope
- Environment configuration analysis
- MCP server setup and validation
- Development tool configuration
- Dependency management
- Security configuration
- Performance optimization

## MCP Tools Used
- **File System Tools**: Analyze configuration files, environment setup, project structure
- **Context7 MCP**: Setup documentation, configuration best practices, tool documentation
- **Brave Search MCP**: Latest setup guides, configuration patterns, tool updates
- **Supabase MCP**: Database configuration, authentication setup, API key management

## Analysis Process

### Phase 1: Environment Assessment
**Current Environment Analysis:**
- Existing configuration validation
- Missing dependencies identification
- Security configuration review
- Performance optimization opportunities

### Phase 2: Configuration Optimization
**Setup Configuration Analysis:**
- MCP server configuration
- Environment variable management
- Tool integration setup
- Development workflow optimization

### Phase 3: Automation Implementation
**Setup Automation Opportunities:**
- Configuration automation
- Setup validation scripts
- Environment consistency checks
- Troubleshooting automation
